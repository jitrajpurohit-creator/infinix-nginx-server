Jitendra's Home Lab Server

Professional Windows Server with IIS Web Hosting

Server Status: Online

This website is hosted on Internet Information Services (IIS) running on Windows Server inside a virtual machine on an Intel Core i3 10th Generation processor.

System Specifications

  • Host Device: Infinix X1 Laptop
  • Host Processor: Intel Core i3 (10th Generation)
  • Host Memory: 8GB RAM
  • Host OS: Windows Server
  • Virtual Machine: Windows Server 2019/2022
  • VM Memory: 2GB RAM (allocated)
  • Web Server: IIS (Internet Information Services)
  • Storage: 40GB dynamic virtual disk
  • .NET Framework: 4.8+

Technology Stack

Windows Server IIS PowerShell HTML5 CSS3 Hyper-V

About This Project

This home lab demonstrates enterprise-grade web server hosting using Windows Server and IIS. The solution showcases professional web hosting capabilities with active directory integration, group policies, security features, and enterprise scalability.

Network Access

To access this website from your host machine:

  • Step 1: Find the VM IP address using PowerShell:
  • ipconfig
  • Step 2: Open your browser and visit:
  • http://<VM-IP-ADDRESS>
  • Example: http://192.168.1.50
