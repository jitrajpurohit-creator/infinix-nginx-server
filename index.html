Jitendra's Home Lab Server

Professional Linux Web Server on VirtualBox

Server Status: Online

This website is powered by Nginx running on Ubuntu Linux inside VirtualBox on an Intel Core i3 10th Generation processor.

System Specifications

  • Host Device: Infinix X1 Laptop
  • Host Processor: Intel Core i3 (10th Gen)
  • Host Memory: 8GB RAM
  • Host OS: Windows 10/11
  • Virtualization Platform: VirtualBox
  • Guest OS: Ubuntu Server 22.04 LTS
  • VM Memory Allocation: 2GB RAM
  • Web Server: Nginx
  • Storage: 20GB dynamic virtual disk

About This Project

This home lab demonstrates how to build a professional web server environment using VirtualBox virtualization. The VM provides complete control over network configuration, storage, and system resources while maintaining efficient performance on modest hardware.

Network Access

To access this website from your host machine:

  • 1. Find the VM IP address: hostname -I
  • 2. Open your browser and visit: http://<VM-IP-ADDRESS>
  • 3. Example: http://192.168.56.101
